# 1.3

## Feladat
Bekér egy nem egész számot, ez lesz a fahrenheit és átváltja celsiusba. 

`C = (F - 32) * 5/9`

## Segédlet
Használjatok `System.out.printf` -et kiiratásnál hogy megfelelő formában jelenjen meg.

Bemenet
```
86
```

```
30.00
```
