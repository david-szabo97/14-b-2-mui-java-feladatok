# 2.2

## Feladat
Bekér egy `N` számot és még `N` számot bekér.

Kiírja az átlagot (két tizedes pontossággal), legnagyobb számot és a legkisebb számot. (Tömb nélkül oldjátok meg)

## Példa #1
Bemenet
```
Mennyi számot akarsz beírni: 3
5
2
7
```

Kimenet
```
Átlag: 4.66
Legnagyobb szám: 7
Legkisebb szám: 2
```
